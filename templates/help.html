Tasks

Summary

The tasks page displays a table with all of your current tasks. Each task shows you the name of the store, the keywords used, the size selected, the profile, the proxies list and the task status. You can also start/stop each task, edit them, duplicate them or delete them.

Creating tasks

  • Store URL ‒ this is the URL of the store you are trying to bot, e.g. https://kith.com
  • Size ‒ this is the size you want for the product. If you don’t care what size, or it’s a one size item then select “Any”
  • Maximum price ‒ if you enter this, then the bot will not purchase the product if its price exceeds this price. Please note that this does not include tax
  • Product selector ‒ this determines how the bot chooses which product to buy. The “Match all keywords” option will only select a product if all of your keywords are found in its name/description. “Highest keyword count” will select the product with the greatest number of keyword matches
  • Keywords ‒ these are the words that will be used to search for the product. Enter them separated by a comma without a space
  • Negative keywords ‒ these words will prevent the bot from buying a product if any of them show up in its name/description. Enter them in the same way as regular keywords
  • Refresh delay ‒ Refresh delay – this is how often the bot will check for updated stock in milliseconds
  • Profile ‒ this is the checkout profile you with to use
  • Proxies ‒ this is the proxy list you want to use
  • Scheduled task ‒ if you would like the bot to start itself at a given time, then input this time here
  • Quantity ‒ this is the number of tasks to create with the selected settings

Captcha

Clicking the "Captcha harvester" button will open the harvester windows. When a task requires a captcha, it will show up here and you will need to solve it for it to proceed.

Profiles

Summary

The profiles page contains a table with all of your created profiles. Each profile shows the name you gave it, the email used, some billing info, last 4 digits of your card and whether or not you selected it to be single checkout. You can also edit each profile, duplicate them or delete them.

Creating a profile

  • Email ‒ this is the email address that will be submitted with your order
  • Phone number ‒ this is the phone number that will be submitted with your order
  • Single checkout ‒ if you only want this profile to checkout once then select this option. Please note that it may not always be possible to cancel other tasks once one has been successful as they may have already started to process the payment
  • Profile name ‒ this is the name of the profile that will show up in the bot
  • Shipping ‒ enter your shipping information
  • Billing ‒ enter your billing information if required
  • Payment ‒ enter your payment information

Proxies

Summary

The proxies page has a table with all of your created proxy lists. Each list shows the name you assigned to it and the amount of proxies in that list. You can also edit each list, duplicate them or delete them.

Creating a proxy list

  • Proxies/span> ‒ enter your proxies into the input in the format IP:port:username:password or IP:port
  • List name ‒ this is the name of the list that will show up in the bot

Account

Summary

The account page displays the email address of your account as well as when your license expires. Additionally, it provides further functions outlined below.

Dashboard

You can visit your dashboard from within the bot, this will link you to our website where you can change things such as your password.

Settings

The “Export Settings” button will save your settings in a JSON file. You can then import this into the bot using the “Import Settings” button and selecting the file.

Log out

When you log out of the bot, it will reset the device activation and so you can use the bot on another device.

Discord webhook

Enter the webhook URL for your server to send checkout details to your Discord. You can enable/disable this with the button, and save your changes with the “Save” button.

Version

This displays the version number of your bot. You can click the “Check for updates” button to ensure that you are up to date.
